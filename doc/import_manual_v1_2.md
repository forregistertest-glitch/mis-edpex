## 1. บทนำ (Introduction)
ในเวอร์ชัน 1.2 ระบบ MIS ได้รับการยกระดับความสามารถในการนำเข้าข้อมูล (Data Import) ให้มีความยืดหยุ่นและแม่นยำยิ่งขึ้น รวมถึงเพิ่มความสามารถในการดึงข้อมูลผลงานตีพิมพ์อัตโนมัติจาก Scopus API เพื่อลดภาระการกรอกข้อมูลของเจ้าหน้าที่

---

## 2. การนำเข้าข้อมูล Excel (Excel Import)

ระบบรองรับการนำเข้าข้อมูลจากไฟล์ Excel (`.xlsx`) โดยมีคุณสมบัติเด่นดังนี้:

### 2.1 Multi-Sheet Support (รองรับหลายชีท)
ท่านสามารถใช้ไฟล์ Excel เพียงไฟล์เดียวที่มีข้อมูลหลายชีทได้ ระบบจะทำการค้นหาข้อมูลโดยอัตโนมัติ:
- **ข้อมูลนิสิต (Students):** ค้นหาจากชีทที่มีคำว่า "รายชื่อ", "Student", "List"
- **ผลงานตีพิมพ์ (Publications):** ค้นหาจากชีทที่มีคำว่า "ผลงาน", "Publication", "Pub"
- **ความก้าวหน้า (Progress):** ค้นหาจากชีทที่มีคำว่า "ความก้าวหน้า", "Progress", "Milestone"

### 2.2 Smart Sheet & Header Detection (ระบบค้นหาอัจฉริยะ)
ไม่จำเป็นต้องตั้งชื่อชีทหรือหัวข้อคอลัมน์ให้ตรงเป๊ะ ระบบใช้ AI Logic ในการตรวจสอบเนื้อหาภายในไฟล์:
- **Flexible Headers:** ระบบเข้าใจชื่อคอลัมน์ที่หลากหลาย เช่น:
  - `Volume` = "ปีที่", "Vol", "VOL"
  - `Issue` = "ฉบับที่", "ISSUE", "Issue"
  - `Pages` = "เลขหน้า", "Pages"
  - `Database` = "ฐานข้อมูล", "Source"
- **Type Safety:** ระบบจะตรวจสอบและแปลงข้อมูลให้อยู่ในรูปแบบที่ถูกต้องก่อนบันทึก ป้องกันข้อผิดพลาดจากการนำเข้า

### 2.3 ขั้นตอนการใช้งาน
1. เข้าไปที่เมนู **"Input Hub"** หรือ **"Academic Dashboard"**
2. คลิกที่ปุ่ม **"Import Data"**
3. เลือกไฟล์ Excel ที่ต้องการ
4. ตรวจสอบพรีวิวข้อมูล และกด **"Confirm Import"**

## 3. การเชื่อมต่อ Scopus (Scopus Auto-Sync)

ฟีเจอร์นี้ช่วยให้ท่านสามารถดึงข้อมูลผลงานตีพิมพ์ล่าสุดของอาจารย์และนิสิตจากฐานข้อมูล Scopus ได้โดยตรง

### 3.1 การทำงาน
- ระบบจะใช้ **API Key** ของมหาวิทยาลัยในการเชื่อมต่อกับ Scopus
- ค้นหาข้อมูลตามชื่อผู้แต่ง (Author Name) หรือ Scopus ID ที่ระบุไว้ในฐานข้อมูลบุคลากร
- ดึงข้อมูล: ชื่อบทความ, วารสาร, ปีที่ตีพิมพ์, Volume, Issue, Pages, และ DOI

### 3.2 ขั้นตอนการใช้งาน
1. เข้าไปที่หน้า **"Academic Dashboard"**
2. คลิกปุ่ม **"Scopus Sync"** (ไอคอน Cloud/Refresh)
3. ระบบจะแสดงสถานะการดึงข้อมูล (Progress Bar)
4. เมื่อเสร็จสิ้น ข้อมูลใหม่จะถูกเพิ่มลงในตารางผลงานตีพิมพ์โดยอัตโนมัติ

> [!IMPORTANT]
> การดึงข้อมูลอาจใช้เวลาสักครู่ขึ้นอยู่กับจำนวนบุคลากรและผลการค้นหา กรุณาอย่าปิดหน้าต่างจนกว่ากระบวนการจะเสร็จสมบูรณ์

---

## 4. การจัดการข้อมูล (Data Management)

### 4.1 การลบข้อมูล (Purge Data)
หากต้องการล้างข้อมูลเพื่อนำเข้าใหม่ทั้งหมด สามารถใช้ปุ่ม **"Purge Data"** ในหน้า Import
- **ระวัง:** การกดปุ่มนี้จะลบข้อมูลที่เลือก (เช่น ผลงานตีพิมพ์ทั้งหมด) ออกจากระบบถาวร ไม่สามารถกู้คืนได้

### 4.2 การแก้ไขข้อมูล
หลังจากนำเข้าหรือ Sync แล้ว ท่านสามารถแก้ไขข้อมูลรายรายการได้โดยคลิกที่ปุ่ม **"Edit"** (ดินสอ) ในตารางแสดงผล
