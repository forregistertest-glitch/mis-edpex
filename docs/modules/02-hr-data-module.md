## 🎯 ภาพรวม

โมดูล HR Data ใช้สำหรับจัดการข้อมูลบุคลากรอย่างครบถ้วน ครอบคลุม **10 tabs** จาก **17 sheets** ของ Excel HR_MIS.xlsx

### **Features หลัก**
- ✅ Form 10 tabs สำหรับข้อมูลบุคลากรครบถ้วน
- ✅ PDPA Data Masking (ซ่อน/แสดงข้อมูลส่วนตัว)
- ✅ Profile Picture Upload
- ✅ ข้อมูลครอบครัว (Family)
- ✅ ประวัติเกียรติคุณ (Honors)
- ✅ ประวัติเงินเดือน (Salary History)
- ✅ ประวัติการลาและอบรม (Leave/Training)

### **ข้อมูลที่ครอบคลุม**
- ข้อมูลส่วนตัวและรูปภาพ
- หมายเลขประจำตัวต่างๆ
- ข้อมูลการติดต่อ
- ข้อมูลการจ้างงาน
- ข้อมูลทางการเงิน
- ประวัติการศึกษา
- ข้อมูลครอบครัว
- ประวัติเกียรติคุณ
- ประวัติเงินเดือน
- ประวัติการลาและอบรม

## ⭐ Features พิเศษ

### **1. PDPA Data Masking**

```typescript
// Toggle PDPA Protection
const [pdpaMasked, setPdpaMasked] = useState(false);

// Mask sensitive data
const maskIdCard = (id: string) => {
  if (!pdpaMasked) return id;
  return id.replace(/(\d)-(\d{4})-(\d{5})-(\d{2})-(\d)/, '$1-****-*****-**-$5');
};
```

**Protected Fields:**
- เลขบัตรประชาชน
- เลขหนังสือเดินทาง
- เลขที่บัญชีธนาคาร (บางส่วน)

### **2. Profile Picture Upload**

```typescript
const [profilePicture, setProfilePicture] = useState<string | null>(null);

const handleImageUpload = (e: React.ChangeEvent<HTMLInputElement>) => {
  const file = e.target.files?.[0];
  if (file) {
    const reader = new FileReader();
    reader.onloadend = () => {
      setProfilePicture(reader.result as string);
    };
    reader.readAsDataURL(file);
  }
};
```

**Features:**
- Preview รูปภาพ
- Upload จาก local
- แสดงรูปเดิม (ถ้ามี)

## 🔧 Technical Details

### **File Structure**

```
src/
├── components/
│   └── data-management/
│       └── hr-data/
│           └── HRDataForm.tsx          # Main form (10 tabs)
├── types/
│   └── data-management.ts              # HRData interface
└── app/
    └── data-management/
        └── hr-data/
            └── new/
                └── page.tsx            # New HR data page
```

### **Component Architecture**

```
HRDataForm
├── State Management
│   ├── formData (HRData)
│   ├── activeTab (string)
│   ├── pdpaMasked (boolean)
│   └── profilePicture (string | null)
├── Handlers
│   ├── handleChange
│   ├── handleSubmit
│   ├── handleImageUpload
│   └── togglePDPA
└── UI Components
    ├── TabButton (10 tabs)
    ├── ProfilePictureUpload
    ├── PDPAToggle
    ├── InputFields
    └── Tables (Education, Family, Honors, etc.)
```

### **Key Implementation**

#### **PDPA Toggle**
```typescript
<button
  type="button"
  onClick={() => setPdpaMasked(!pdpaMasked)}
  className={`... ${pdpaMasked ? 'bg-emerald-600' : 'bg-gray-300'}`}
>
  🔒 PDPA: {pdpaMasked ? 'ซ่อนข้อมูลส่วนตัว' : 'แสดงข้อมูลทั้งหมด'}
</button>
```

#### **Profile Picture**
```typescript
<div className="relative w-32 h-32 mx-auto">
  {profilePicture ? (
    <img src={profilePicture} alt="Profile" className="..." />
  ) : (
    <div className="...">
      <User size={48} />
    </div>
  )}
  <input type="file" accept="image/*" onChange={handleImageUpload} />
</div>
```
