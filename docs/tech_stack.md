# ⚙️ Tech Stack & Architecture

## Core Technologies
- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS + PostCSS
- **Database / Backend:** Firebase (Firestore, Auth)

## Key Project Structures
- `src/app/`: Next.js App Router endpoints and primary layout.
- `src/components/`: Reusable React components grouped by feature (e.g., `personnel`, `import-data`).
- `src/services/`: Modules handling data interactions with Firebase (e.g., `personnelService.ts`).
- `src/contexts/`: React Contexts for global state management (e.g., Authentication).
- `src/types/`: TypeScript definitions and generic interfaces for data models.
- `docs/`: Project documentation and style guidelines.
